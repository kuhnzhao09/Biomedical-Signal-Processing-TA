# Course Blueprint Template

Use this file when the request is about course planning, syllabus design, lesson sequencing, or aligning exercises with learning outcomes.

Replace the generic outline below with the local course structure when course materials become available.

## Recommended Module Sequence

1. Signals, systems, and biomedical acquisition context
2. Sampling, quantization, aliasing, and ADC constraints
3. Time-domain statistics and descriptive features
4. Fourier analysis, spectral leakage, and windowing
5. Digital filters: FIR, IIR, notch, band-pass, and stability
6. Time-frequency methods: STFT and wavelets
7. Event detection and segmentation
8. Feature extraction and dimensionality reduction
9. Pattern recognition or introductory machine learning for signals
10. Integrated case studies using ECG, EMG, EEG, and PPG

## Learning Outcome Template

For each module, define:

- What students should explain conceptually
- What students should compute by hand or derive
- What students should implement in MATLAB or Python
- What plots or metrics they should interpret
- What common failure modes they should avoid

## Lesson Planning Pattern

Use this sequence for a 90-minute class:

1. Start with a physiological or engineering problem.
2. Introduce the signal-processing concept.
3. Show one equation or algorithmic core.
4. Walk through one small numerical example.
5. Connect the method to a real biomedical signal.
6. End with a quick check question or mini-lab.

## Material Authoring Checklist

- State the signal type and acquisition assumptions.
- Define all symbols before using formulas.
- Keep units explicit: Hz, ms, mV, uV.
- Pair every new method with at least one failure case.
- Distinguish preprocessing, feature extraction, and interpretation.
