# Quick Start

## 1. Use the system prompt

Copy the contents of `system-prompt.md` into the system prompt or agent instructions field of your assistant platform.

## 2. Attach the current materials

Use the `materials/` folder as the initial knowledge source.

## 3. Invoke the skill behavior

When working inside Codex, refer to the skill `$biomedical-signal-processing-teaching-assistant` for teaching behavior, lab guidance, and assignment boundaries.

## 4. Start with these tasks

Good first tasks for this generic agent:

- Explain sampling, aliasing, FFT, and digital filtering.
- Compare ECG and PPG preprocessing pipelines.
- Give hint-only help for a MATLAB homework question.
- Review a short Python script for PSD or filtering mistakes.
- Draft a basic undergraduate lab on EMG envelope extraction.

## 5. Upgrade path

Later, improve the agent by adding:

- your course outline,
- your preferred formulas and notation,
- your own MATLAB and Python examples,
- your lab sheets,
- your grading rules.
